GPU accelerated collective Monte Carlos methods
---------------------------------------------------------------------


This folder contains the code for the arXiv preprint `Collective Proposal Distributions for Nonlinear MCMC samplers: Mean-Field Theory and Fast Implementation <https://arxiv.org/pdf/1909.08988.pdf>`_.
